# Правила проекта CRISHA.RU

> **CRITICAL**: Перед выполнением ЛЮБОЙ задачи обязательно прочитать скилл `project-rules`:
> `d:\CRISHA.RU\.agents\skills\project-rules\SKILL.md`

## Краткие правила

1. **ОБЯЗАТЕЛЬНО** читать скилл `project-rules` перед любой задачей
2. **Язык**: строго Русский (общение + комментарии в коде)
3. **Подтверждение**: начинать каждый ответ с `✅ Правила прочитаны и приняты`
4. **Стиль**: придерживаться существующих паттернов в проекте
5. **Этапность**: после планирования делить задачу на этапы и спрашивать разрешения перед каждым новым этапом
6. **AutoRun**: выставлять `SafeToAutoRun: true` для безопасных операций (чтение файлов, запуск серверов, установка зависимостей и т.д.) — не прерывать работу лишними подтверждениями
